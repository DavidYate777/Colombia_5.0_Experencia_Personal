Categorías: Cybersecurity UI / Game Design General Tech
Categoría Término (EN) Español Definición Example in English
Cybersecurity Cybersecurity Ciberseguridad Conjunto de prácticas para proteger sistemas, redes y datos de ataques digitales. "The company invested in cybersecurity to protect its users' data."
Cybersecurity Firewall Cortafuegos Sistema que monitorea y controla el tráfico de red según reglas predefinidas. "A firewall blocks unauthorized access to a private network."
Cybersecurity Encryption Cifrado Proceso de convertir datos en un formato ilegible para proteger su confidencialidad. "End-to-end encryption ensures only the sender and receiver can read the message."
Cybersecurity Ransomware Secuestro de datos Malware que cifra los archivos de la víctima y exige pago para restaurar el acceso. "The hospital was attacked by ransomware and lost access to patient records."
Cybersecurity Phishing Suplantación de identidad Técnica para engañar a usuarios y obtener credenciales o información sensible. "The phishing email looked real but was designed to steal passwords."
Cybersecurity Zero Trust Confianza cero Modelo de seguridad que no confía en ningún usuario o dispositivo por defecto. "Zero trust means every user must verify their identity, even inside the network."
Cybersecurity Vulnerability Vulnerabilidad Debilidad en un sistema que puede ser explotada por un atacante. "The developer patched the vulnerability before it could be exploited."
Cybersecurity Data Breach Brecha de datos Incidente donde información confidencial es accedida sin autorización. "The data breach exposed millions of users' email addresses."
Cybersecurity Penetration Testing Prueba de penetración Simulación controlada de ataques para identificar vulnerabilidades en sistemas. "They hired a team to run penetration testing before launching the app."
Cybersecurity SOC Centro de Operaciones de Seguridad Equipo centralizado que monitorea y responde a incidentes de seguridad en tiempo real. "The SOC team detected the intrusion within minutes."
Cybersecurity Intrusion Detection Detección de intrusos Sistema que identifica actividades sospechosas o maliciosas en la red. "Intrusion detection systems alert admins when unusual traffic is found."
Cybersecurity Threat Analysis Análisis de amenazas Proceso de identificar y evaluar posibles amenazas a la seguridad de un sistema. "Threat analysis helps companies prepare for potential cyberattacks."
UI / Game UI Design Diseño de interfaz Proceso de crear interfaces visuales para una interacción efectiva con el usuario. "Good UI design makes the app easy and enjoyable to use."
UI / Game UX Experiencia de usuario Conjunto de aspectos relacionados con la interacción del usuario con un producto digital. "The UX team tested the app with real users to improve the flow."
UI / Game HUD Pantalla superpuesta Elementos visuales mostrados sobre la pantalla en videojuegos (salud, munición, mapa). "The HUD shows the player's health and ammo without blocking the view."
UI / Game Wireframe Prototipo esquemático Boceto de baja fidelidad que muestra la estructura básica de una interfaz. "We built a wireframe to plan the layout before writing any code."
UI / Game Prototype Prototipo Versión preliminar de una interfaz usada para probar conceptos antes del desarrollo final. "The prototype let users test the game menu before it was fully built."
UI / Game Vector Vector Elemento gráfico definido matemáticamente, escalable sin pérdida de calidad. "Icons made with vectors look sharp on any screen size."
UI / Game Design System Sistema de diseño Colección de componentes reutilizables para mantener consistencia visual en un producto. "The design system kept all buttons and colors consistent across the app."
UI / Game Component Componente Unidad reutilizable de interfaz que encapsula estructura, estilo y comportamiento. "The navigation bar is a component reused on every page."
UI / Game Responsive Design Diseño responsivo Técnica que adapta el contenido web a diferentes tamaños de pantalla. "Responsive design makes the website look good on phones and desktops."
General Tech Artificial Intelligence Inteligencia Artificial Rama de la informática que desarrolla sistemas capaces de realizar tareas que requieren inteligencia humana. "Artificial intelligence is being used to detect diseases in medical images."
General Tech Machine Learning Aprendizaje automático Subcampo de la IA que permite a los sistemas aprender con la experiencia sin programarse explícitamente. "The spam filter uses machine learning to improve over time."
General Tech Cloud Computing Computación en la nube Servicios de computación ofrecidos a través de internet sin infraestructura local. "We store all our files in the cloud so the team can access them anywhere."
General Tech API Interfaz de programación Conjunto de protocolos que permiten la comunicación entre diferentes aplicaciones. "The weather app uses an API to get real-time data from a server."
General Tech DevOps Desarrollo y Operaciones Conjunto de prácticas que une desarrollo de software y operaciones IT para acelerar entregas. "DevOps practices helped the team deploy updates every day instead of every month."
General Tech Automation Automatización Uso de tecnología para realizar tareas con mínima intervención humana. "Automation reduced the time to process orders from hours to seconds."
General Tech Data Privacy Privacidad de datos Derecho de los individuos a controlar cómo se recopila y usa su información personal. "The app asks for permission before collecting any personal data."
General Tech Agile Ágil Metodología de desarrollo basada en iteraciones cortas, colaboración y adaptabilidad. "The agile team released a new feature every two weeks."
General Tech Scrum Scrum Marco de trabajo ágil que organiza el desarrollo en sprints con roles y eventos definidos. "Every morning the scrum team had a short meeting to review progress."
General Tech Frontend Interfaz del usuario Parte visible de una aplicación con la que el usuario interactúa directamente. "The frontend developer built all the buttons, menus and animations."
General Tech Backend Servidor Parte del software que opera en el servidor, manejando lógica, base de datos y autenticación. "The backend stores user data and handles login security."
General Tech Open Source Código abierto Software cuyo código fuente es público y puede ser libremente usado, modificado y distribuido. "Linux is an open source operating system used by millions of servers."
General Tech GitHub Pages GitHub Pages Servicio gratuito de GitHub para publicar sitios web estáticos desde repositorios. "This website is hosted for free using GitHub Pages."
