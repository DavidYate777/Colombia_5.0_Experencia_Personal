Términos técnicos

Inglés / English Español Definición
CybersecurityCiberseguridadConjunto de prácticas y tecnologías para proteger sistemas, redes y datos de ataques digitales.
FirewallCortafuegosSistema de seguridad que monitorea y controla el tráfico de red entrante y saliente según reglas predefinidas.
EncryptionCifradoProceso de convertir datos en un formato ilegible para proteger la confidencialidad de la información.
RansomwareSecuestro de datosTipo de malware que cifra los archivos de la víctima y exige un pago para restaurar el acceso.
PhishingSuplantación de identidadTécnica de ingeniería social para engañar a usuarios y obtener credenciales o información sensible.
Zero TrustConfianza ceroModelo de seguridad que no confía en ningún usuario o dispositivo por defecto, dentro o fuera de la red.
Threat AnalysisAnálisis de amenazasProceso de identificar, evaluar y priorizar posibles amenazas a la seguridad de un sistema.
Data BreachBrecha de datosIncidente en el que información confidencial es accedida, robada o expuesta sin autorización.
VulnerabilityVulnerabilidadDebilidad en un sistema o software que puede ser explotada por un atacante para causar daño.
SOCCentro de Operaciones de SeguridadEquipo centralizado que monitorea y responde a incidentes de seguridad en tiempo real.
Penetration TestingPrueba de penetraciónSimulación controlada de ataques para identificar vulnerabilidades en sistemas antes que actores maliciosos.
Intrusion DetectionDetección de intrusosSistema que monitorea la red para identificar actividades sospechosas o maliciosas.
UI DesignDiseño de interfazProceso de diseñar interfaces visuales de aplicaciones o sitios web para una interacción efectiva.
UXExperiencia de usuarioConjunto de aspectos relacionados con la interacción del usuario con un producto o servicio digital.
HUDPantalla superpuestaElementos visuales mostrados sobre la pantalla principal en videojuegos (salud, munición, mapa).
WireframePrototipo esquemáticoBoceto de baja fidelidad que representa la estructura y distribución de una interfaz.
PrototypePrototipoVersión preliminar de un producto o interfaz usada para probar conceptos antes del desarrollo final.
VectorVectorElemento gráfico definido matemáticamente, escalable sin pérdida de calidad.
Design SystemSistema de diseñoColección de componentes reutilizables, guías y estándares para mantener consistencia visual.
ComponentComponenteUnidad reutilizable de interfaz que encapsula estructura, estilo y comportamiento.
Artificial IntelligenceInteligencia ArtificialRama de la informática que desarrolla sistemas capaces de realizar tareas que requieren inteligencia humana.
Machine LearningAprendizaje automáticoSubcampo de la IA que permite a los sistemas aprender y mejorar automáticamente con la experiencia.
Cloud ComputingComputación en la nubeServicios de computación (servidores, almacenamiento, bases de datos) ofrecidos a través de internet.
APIInterfaz de programaciónConjunto de protocolos y herramientas que permiten la comunicación entre diferentes aplicaciones de software.
DevOpsDesarrollo y OperacionesConjunto de prácticas que une desarrollo de software y operaciones IT para acelerar entregas.
AutomationAutomatizaciónUso de tecnología para realizar tareas con mínima intervención humana, mejorando eficiencia.
Data PrivacyPrivacidad de datosDerecho de los individuos a controlar cómo se recopila, usa y comparte su información personal.
AgileÁgilMetodología de desarrollo de software basada en iteraciones cortas, colaboración y adaptabilidad.
ScrumScrumMarco de trabajo ágil que organiza el desarrollo en sprints con roles y eventos definidos.
FrontendInterfaz del usuarioParte visible de una aplicación web o móvil con la que el usuario interactúa directamente.
BackendServidorParte del software que opera en el servidor, manejando lógica, base de datos y autenticación.
Responsive DesignDiseño responsivoTécnica de diseño web que adapta la presentación del contenido a diferentes tamaños de pantalla.
Open SourceCódigo abiertoSoftware cuyo código fuente es público y puede ser libremente usado, modificado y distribuido.
GitHub PagesGitHub PagesServicio de alojamiento estático gratuito de GitHub para publicar sitios web directamente desde repositorios.
